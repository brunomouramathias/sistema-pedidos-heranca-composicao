# Instruções para Publicar no GitHub

## Passo 1: Inicializar o repositório local

```bash
cd C:\Users\BRUNO\PedidosVenda
git init
git add .
git commit -m "Implementação inicial: Sistema de Pedidos com Herança e Composição"
```

## Passo 2: Criar repositório no GitHub

1. Acesse https://github.com/new
2. Nome sugerido: `sistema-pedidos-heranca-composicao`
3. Descrição: "Sistema de processamento de pedidos nacionais e internacionais usando herança controlada e composição em C#"
4. Deixe como público
5. NÃO inicialize com README (já temos um)
6. Clique em "Create repository"

## Passo 3: Conectar e fazer push

```bash
git remote add origin https://github.com/SEU_USUARIO/sistema-pedidos-heranca-composicao.git
git branch -M main
git push -u origin main
```

## Passo 4: Verificar

Acesse seu repositório no GitHub e confirme que todos os arquivos estão lá.
